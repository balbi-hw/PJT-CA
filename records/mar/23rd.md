# 23RD MAR

`WordCard` 클래스와 `CardMain` 메인 메서드를 만들었습니다.
  
잠깐이지만 `private` 과 `static` 을 혼동해서 값이 의도대로 저장되지 않았던 해프닝도 있었고
  
생성자 오버로딩을 사용해봤습니다. 카드에서 단어는 필수 요소이지만
뜻과 예문은 선택 요소이기 때문에 따로 저장할 수 있게 했습니다.
  
OOP 를 위해 나름대로 역할과 기능을 분리해봤고 어느정도 마음에 듭니다.
  
메인 메서드의 카드 생성은 수동으로 직접 입력했지만 입력을 받아서 만드는 것도 문제 없어보입니다.
  
간단하게 만들어보았지만 나중에는 실행시 [카드 생성, 카드 출력, 예문 입력] 등의 메뉴를 출력하고
사용자의 선택에 따라 다른 출력을 내도록 해도 좋을 것 같습니다.  
  
예문 입력은 카드 출력 단계에서 해도 될 것 같고 카드 출력 또한
단어 목록을 출력하고 그 중 단어를 선택하면 그 안의 필드들을 출력하도록 해도 좋을 것 같습니다.
  
데이터베이스는 아직 잘 모르지만 나중에 틀린 횟수를 객체에 저장할 수 있게 하면 지금까지 누적 틀린 횟수를 확인 할 수 있을 것 같습니다.

  
  
정말 간단한 기능들만 구현했는데 for문 - if문 - if문 으로 조금 길어졌습니다.
이것들도 다 메서드로 기능 구분해보겠습니다.
