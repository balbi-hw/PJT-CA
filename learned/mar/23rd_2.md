3. 클래스 생성
    ```java
    class card {
        static String word;
        static String meaning;
        static String example;
    
        card(String word, String meaning, String example) {
            this.word = word;
            this.meaning = meaning;
            this.example = example;
        }
    
        public void printCard() {
            System.out.println("단어: " + word);
            System.out.println("뜻: " + meaning);
            System.out.println("예문: " + example);
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            card card = new card("apple", "사과", "I eat an apple every day.");
            card.printCard();
        }
    }
    ```
    구상했던 필드를 구현하고 생성자와 출력 메서드를 구현했다.

4. 정답 판정
```java
import java.util.Scanner;

class card {
    static String word;
    static String meaning;
    static String example;

    card(String word, String meaning, String example) {
        this.word = word;
        this.meaning = meaning;
        this.example = example;
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        card card = new card("apple", "사과");

        System.out.print("다음 단어의 뜻을 입력하세요: " + card.word);
        String answer = input.nextLine();
        
        if (answer.equals(card.meaning)) {
            System.out.println("정답입니다.");
        } else {
            System.out.println("오답입니다. 정답은 " + card.meaning + " 입니다.");
        }
    }
}
```
메인 메서드를 구현했다. 카드를 제시하고 단어를 입력 받은 뒤 정답 여부를 판단한다.
