# Anki Clone

Anki 스타일의 반복 학습 카드 앱을 구현하는 프로젝트입니다.  
카드 생성, 조회, 복습 기능을 중심으로 학습용 서비스를 직접 개발합니다.
  
단순히 기능을 따라 만드는 것에 그치지 않고,  
카드 관리, 복습 흐름, 이미지 검색 연동 등의 기능을 직접 설계하고 구현하며  
실제 서비스 개발 과정을 경험하는 것을 목표로 합니다.
  
또한 본인의 실력이 부족하다는 점을 인지하고 있어 프로젝트 중 AI를 학습에 적극 활용하여,
본인의 AI 활용 역량을 기르고, 모르는 내용을 학습할 때 AI를 활용하는 연습을 하는데에도 의의가 있습니다.

## Just out of curiosity
해당 프로젝트는 자바와 스프링 부트를 공부하던 중 강의만 듣는 **수동적 학습**에서 벗어나
배운 내용들을 직접 사용해보고 싶어서 기획하게 되었습니다.  
  
'본인이 사용할 프로그램을 처음부터 끝까지 혼자 완성해보는 경험' 은 누구에게 물어도 항상
중요하다고 하는 것이기에 이를 위한 프로젝트이기도 합니다.  
  
Anki 프로그램은 매일매일 사용하고 있으며 추가했으면 좋을 것 같은 기능도 있었기에
프로젝트 주제로 적합하다고 생각했습니다.
  
직접 기능을 설계하고 프론트엔드·백엔드·데이터베이스를 연결하며  
풀스택 개발 흐름을 익히는 데 초점을 두었습니다.

## Tech Stack

### Frontend
- Thymeleaf ( 추후 전환 예정 )

### Backend
- Java
- Spring Boot

### Database
- H2

### Tools
- Git
- GitHub
- REST API

## Features
- 카드 생성 / 수정 / 삭제
- 카드 복습
- 반복 학습 기반 복습 흐름
- 이미지 검색 연동 기능

## Goal
- Anki와 유사한 핵심 기능 구현
- 프론트엔드/백엔드/DB 연동 경험
- 클라이언트-서버 구조에 대한 이해
- 데이터베이스 설계 및 API 연동 경험 축적
- 기능 구현 중심의 클론 프로젝트를 통한 실전 감각 향상
- AI 활용 역량 강화
